Setup instructions:

1. Install python 3.6.2 or above (LINK:https://www.python.org/downloads/)
2. Input the data in the data folder save as node.csv and edge.csv (FILENAMES SHOULD NOT BE CHANGED)
3. If your OS is Windows double click "Win_RunGen.bat" , If your OS is Linux or Mac run "./Win_RunGen.sh" in terminal
4. Check output folder for the generated output
5. To view the output go to "https://www.yworks.com/yed-live/"
6. Select Open an Existing Document then select From Local File
7. Select the generated output file and open
8. Select the Layout -> Balloon and click Apply.
